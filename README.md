## 💻 자주 사용하는 Java 문법  
- **Stream 활용**: 데이터를 효율적으로 처리하는 함수형 프로그래밍  
- **Collections API**: List, Set, Map 등 활용한 문제 해결  
- **Lambda 표현식**: 간결하고 읽기 쉬운 코드 작성  
- **BigInteger**: 큰 정수 계산에 필요한 경우
